
# About This Project
This project allows creating chat rooms and allowing clients to message each other in these chat rooms.


## Clone This Poject

```bash
  git clone https://github.com/HarunBulbull/socket-io-chat.git
  cd socket-io-chat
  npm i
  cd socket
  npm i
```

## Run This project
(!) 2 Terminal required.
```bash
  cd socket-io-chat
  npm run dev
  cd socket-io-chat/socket
  npm start
```
###

Enjoy!
